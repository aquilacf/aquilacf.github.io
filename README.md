# aquilafreitas.com
Front end application for aquilafreitas.com.

## Requirements
- [Node.js >= 0.8.0](http://nodejs.org/)
- [ExpressJS >= 4.15.4](https://expressjs.com/)
- [Nodemon >= 0.8.0](https://nodemon.io/)

## Installation
- Checkout from Git.
- Run `npm install`.
- Run `npm run dev` to start a development server.
